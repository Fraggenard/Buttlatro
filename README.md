# Buttlatro
A mod to add [Buttplug.io](https://buttplug.io/) support for vibration-based [compatible devices](https://iostindex.com/?filter0ButtplugSupport=4) to Balatro! Sends vibration commands to your device that increase in intensity over time as your played hand is being scored. Requires [Balamod](https://github.com/balamod/balamod) and [Intiface Central](https://intiface.com/central/)

## Installation
1. Install [Balamod](https://github.com/balamod/balamod)
2. Install [Intiface Central](https://intiface.com/central/) and connect your device
3. Download and extract Buttlatro zip folder and place into Balamod `mods` folder  
   - **Windows**: `C:\Users\<username>\AppData\Roaming\Balatro`
   - **macOS**: `~/Library/Application Support/Balatro`
   - **Linux**: `~/.local/share/Steam/steamapps/compatdata/2379780/pfx/drive_c/users/steamuser/AppData/Roaming/Balatro`
4. Ensure Buttlatro is enabled in the Mods menu. If disabled, restart Balatro after enabling
   ![2379780_28](https://github.com/user-attachments/assets/d4689312-efcd-451b-8729-30e468508b83)
5. Play Balatro with your Intiface server running!

## Usage
While in-game, press `m` to open the test vibration menu to check everything is working.
